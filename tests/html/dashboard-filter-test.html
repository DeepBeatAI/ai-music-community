✅ Dashboard Filter Fix Applied

Issue Fixed: Filters now apply to existing posts on the dashboard page, not just search results.

What Was Fixed

Problem: The dashboard filters (Content Type, Sort By, Time Range) were only working when performing a search. When viewing all posts without searching, the filters had no effect.

Solution: Modified the applyFiltersAndSearch function to always check for active search filters, even when there's no active search query. This allows users to filter existing posts using the filter controls.

Key Changes Made:
1. Updated filter detection logic to work without requiring an active search
2. Fixed post type filtering to handle 'creators' filter properly
3. Updated React dependencies to trigger re-filtering when search filters change
